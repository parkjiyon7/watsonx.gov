# ML 모델 평가 (ML Model Evaluation)

## 개요
이 가이드는 watsonx.gov on-premise 환경에서 머신러닝 모델을 평가하는 방법을 설명합니다.

## 튜토리얼 참조
- **공식 문서**: [머신러닝 모델 평가](https://www.ibm.com/docs/en/watsonx/w-and-w/2.3.x?topic=ai-evaluate-machine-learning-model)

## 사용 파일
- **프로젝트 파일**: `evaluate-an-ml-model`
  - 이 디렉토리에는 다음과 같은 필수 자산이 포함되어 있습니다:
    - 학습된 ML 모델
    - 평가용 데이터셋
    - 노트북 및 스크립트
    - 모델 구성 파일

## 사전 요구사항
- watsonx.ai 플랫폼 접근 권한
- Watson OpenScale 설정
- 평가할 배포된 ML 모델
- 평가용 홀드아웃 데이터
