☁️ SaaS 환경

클라우드 환경에서 watsonx.governance를 사용하는 실습 가이드입니다.

준비 중 SaaS 환경 실습 가이드는 현재 준비 중입니다.

🚀 시작하기

1단계: 환경 준비

  • watsonx.ai 플랫폼에 로그인
  • 필요한 프로젝트 파일 다운로드
  • Python 환경 설정 (Jupyter Notebook)

2단계: 실습 진행

  • 상세 가이드 문서를 참조하여 단계별로 진행
  • 각 단계의 결과를 확인하며 학습
  • 문제 발생 시 Troubleshooting 페이지 참조

3단계: 결과 확인

  • 평가 결과 분석
  • 대시보드에서 메트릭 확인
  • 리포트 생성 및 공유
